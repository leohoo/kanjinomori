# .dev - Development Notes

This folder contains project management files for development tracking.

## Structure

```
.dev/
├── specs/      # Project specifications
├── tasks/      # Task descriptions and requirements
├── reports/    # Progress reports and session logs
└── data/       # Data source documentation
```

## Naming Convention

### Tasks
`NNN-short-description.md` (e.g., `001-initial-implementation.md`)

### Reports
`YYYY-MM-DD.md` (e.g., `2024-11-29.md`)
